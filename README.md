# 📊 Business Insights 360 — Power BI Dashboard

An end-to-end BI report for **AtliQ Technologies** covering Finance, Sales, Marketing, Supply Chain, and Executive insights in a single interactive dashboard.
## 📄 Report Pages

| Page | Description |
| 🏠 Home | Navigation landing page |
| 📋 P&L Check | Actuals vs estimates breakdown |
| 💰 Finance View | P&L with YoY trends and market/segment cuts |
| 🛒 Sales View | Customer & product performance — NS$, GM% |
| 📣 Marketing View | Segment profitability — GM% and NP% analysis |
| 🚚 Supply Chain View | Forecast Accuracy, Net Error, ABS Error |
| 📈 Sales Trend | Time-series sales trend |
| 🏢 Executive View | High-level KPIs — Revenue, GM%, NP%, Market Share |

## 🔑 Key Metrics

`NS $` · `GM %` · `Net Profit %` · `Forecast Accuracy %` · `Net Error` · `Market Share %` · `RC %`

All metrics support **YoY comparison** and **benchmark toggle** (Last Year / Target).

## 🛠️ Tech Stack
**Power BI Desktop** · **DAX** · **Power Query** · **Star Schema**
